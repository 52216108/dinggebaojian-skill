# 订个包间 Skill

让 AI 助手能实时查询全球合作餐厅的包间信息、VR 全景、空闲库存等。

基于 [MCP (Model Context Protocol)](https://modelcontextprotocol.io/) 协议，支持所有兼容 MCP 的 AI 助手。

## 功能

| 能力 | 说明 |
|---|---|
| 一键搜索包间 | 按城市、人数、设施一次性筛选空闲包间 |
| 餐厅信息查询 | 地址、电话、营业时间、菜系、人均消费、停车指引 |
| 实时包间库存 | 查询指定日期和餐段的空闲包间 |
| VR 全景看房 | 每个包间可 360° 在线查看 |
| 智能识别店名 | 所有工具支持直接传店名，无需先查编码 |

## 安装

复制下面这句话，发给你的 AI 助手：

> 帮我安装订个包间 Skill，技能地址：https://gitee.com/dinggebaojian/dinggebaojian-skill

或在 MCP 配置中手动添加：

```json
{
  "mcpServers": {
    "dinggebaojian-skill": {
      "type": "streamable-http",
      "url": "https://mcp.dinggebaojian.com/mcp"
    }
  }
}
```

**无需任何认证**，直接连接即可使用。

支持所有兼容 MCP 协议的 AI 助手。
